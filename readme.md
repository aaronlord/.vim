### Installation

Install [Neovim](https://github.com/neovim/neovim).

```
$ cd ~
$ git clone https://github.com/aaronlord/.vim.git
$ rm -rf ~/.config/nvim
$ ln -s ~/.vim ~/.config/nvim
$ nvim
```

Be sure to run `:checkhealth` dependancy issues (node, python, php, etc).
